jsDelivr JavaScript的CDN - 先进的开源公共CDN

作为开发人员，你可能意识到谷歌托管库。谷歌提供了一个方便， 快速的方法，包括在你的网站14流行的javascript库。

但是，如果你是一个站长，你想利用一个快速的cdn与其他较冷门的项目吗？或者，如果你是 开发人员，你想使你的项目更容易访问和其他用户使用？

这是 jsdelivr进场。 jsdelivr cdn是为了帮助开发人员和网站管理员一个自由和开放源码。 有没有普及的限制和各种文件都是允许的，包括javascript库，jquery插件，css框架，字体等等。
